# 💰 Personal Finance ML

> CSV 기반 개인 재무 데이터 자동 정제 및 분석 시스템  
> From Raw Transactions → Clean Data → Analytics → ML → Dashboard

---

## 🚀 Project Goal

이 프로젝트는 개인 금융 데이터를 구조화하고,  
자동 정제 및 분석을 통해 소비 패턴을 이해하고  
향후 지출을 예측하는 ML 기반 시스템입니다.

단순 가계부가 아니라,  
**데이터 파이프라인 + 분석 + 예측 모델**까지 설계합니다.

---

## 🧱 System Architecture

Raw CSV  
→ Data Cleaning  
→ Feature Engineering  
→ Monthly Aggregation  
→ Analytics  
→ ML Prediction  
→ Streamlit Dashboard (Future)
